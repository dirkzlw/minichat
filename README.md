# minichat
微聊的服务端  

开发工具：IntelliJ IDEA  

所用技术：
    1、Spring Boot + Spring Data JPA + Netty  
    2、Mysql数据库 + Druid数据库连接池  
    3、FastDFS文件服务器  
    4、Jasypt加密技术  

完成功能：  
    1、用户注册  
    2、用户登录  
    3、用户修改头像  
    4、用户修改昵称  
    5、完成用户搜索好友  
    6、完成用户发送好友请求  
    7、完成用户请求列表的展示  
    8、完成用户"忽略/通过"好友请求
